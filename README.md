# Zomato Orders Analysis – SQL & Power BI

**End-to-end analysis of Zomato order data** using SQL and Power BI to extract actionable insights on top dishes, average order value, add-ons, offers.

---

## 🔹 Tools Used
- **SQL:** Joins, aggregations, subqueries for in-depth analysis  
- **Power BI:** Interactive dashboards and KPI visualizations  
- **Excel (optional):** Data cleaning & preprocessing  

---

## 🔹 Key Highlights
- Identified **top most ordered dishes**  
- Visualized **Avg Order Value trends and seasonality**  
- Measured **add-on & offer effectiveness**  
- Interactive dashboard for **quick decision-making**
