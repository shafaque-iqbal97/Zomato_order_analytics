# Zomato Orders Analysis – SQL & Power BI

**End-to-end analysis of Zomato order data** using SQL and Power BI to extract actionable insights on top dishes, average order value, add-ons, offers.

---

## 🔹 Tools Used
- **SQL:** Joins, aggregations, subqueries for in-depth analysis  
- **Power BI:** Interactive dashboards and KPI visualizations  
- **Excel (optional):** Data cleaning & preprocessing  

---

## 🔹 Key Highlights
- Identified **top 5 most ordered dishes**   
- Visualized **Average Order Value (AOV) trends**
- Measured **add-on & offer effectiveness** over time  
 - Determined **most popular dishes and add-ons**  
- Tracked **revenue distribution across categories**  
- Created **interactive KPIs** for quick business decision-making  
- Built **filters and slicers** in Power BI for dynamic analysis  
- SQL queries include **aggregations,group by,order by, and top-n analyses**  

---

## 🔹 How to Use
1. Open `Zomato_Dashboard.pbix` in Power BI  
2. Run `analysis_queries.sql` in any SQL environment  
3. Dataset located at `zomato_orders.csv`
